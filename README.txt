# 🎯 Attention Drift Detector

AI that detects when you lose focus while typing - using only keystroke dynamics. No camera, no microphone, no special hardware.

## 📊 Results
- **Accuracy:** 71.4%
- **Focused Detection:** 90%
- **Model:** Random Forest
- **Data:** 215 keystrokes
