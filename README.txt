## 🎯 What is this?

An AI system that detects when you lose focus while typing - using only keystroke dynamics. No camera, no microphone, no special hardware.

Built as my first independent research project during 2nd semester of BTech CSE (AI & ML) at Graphic Era University, Dehradun.

## 📊 Results

| Metric | Value |
|--------|-------|
| Accuracy | 71.4% |
| Focused Detection | 90% |
| Distracted Detection | 25% |
| Model | Random Forest |
| Training Data | 215 keystrokes |
